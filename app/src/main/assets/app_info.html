应用信息

DNSSR // LOCAL RESOLVER

LOCAL DNS CONTROL PLANE

DNSSR 本地解析控制台

在 Android 设备本机管理 DNS 查询路径，让解析策略、缓存、规则与上游连接处于可控状态。

版本 --Android VpnService本地优先

核心能力

01 / CAPABILITIES
[01]

加密上游

支持 DNS、DoH、DoH3 与 DoT，并可管理自定义服务商。

[02]

规则与缓存

以缓存、屏蔽规则、白名单和订阅规则减少重复请求与干扰。

[03]

可观测性

通过请求日志、竞速统计、服务商健康和 Bootstrap 数据追踪状态。

[04]

独立引导

支持 Bootstrap IP，降低解析加密上游域名时对系统 DNS 的依赖。

[05]

配置流转

支持自定义服务、规则订阅的导入导出与自动更新。

[06]

快捷控制

提供系统快捷设置磁贴，方便快速启停 DNS 本地通道。

运行边界

02 / ARCHITECTURE

仅处理 DNS

DNSSR 通过 Android VpnService 建立仅处理 DNS 的本地通道，不代理普通网页、应用数据或文件传输流量。启用时 Android 会请求 VPN 权限。

实际解析表现

解析速度、稳定性和可用性取决于网络环境、所选上游和本机规则。DoH3 基于 QUIC，需要网络允许 UDP/443。

隐私与本地存储

缓存、请求日志、规则与服务商配置保存在设备本机。上游仍会收到必要查询；仅 DoH、DoH3、DoT 会加密到上游的 DNS 传输。
